# obsidian-python

1. Create a repository.
2. Create a personal access token.
3.  Install the Obsidian Git community plugin
4. Create a folder to store the repository. (e.g. remote-blog/).
5. Run the command (CMD/Ctrl + P): Clone an existing remote repo clone-repo-git-plugin.png
6. Paste the URL of the forked repository in the following format https://<PERSONAL_ACCESS_TOKEN>@github.com/<USERNAME>/<REPO>.git
7. Then type in the folder you created in step 5 (e.g. remote-blog/)
8. Restart Obsidian
9. Make edits to your notes
10. Publish your notes run the command “Obsidian Git: Create backup” by opening the command palette (CMD/Ctrl + P)
